### Instalação

1. Clonar o repositório
   ```sh
   git clone https://github.com/lucasamadeuu/A.I-Gym.git
   ```
2. Instalar pacotes NPM
   ```sh
   npm install
   ```
   
3. Iniciar o projeto
   ```sh
   npm run dev
   ```
